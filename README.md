# 叠印元素 (Overlay Elements)

## 项目概述
《叠印元素》是一款轻竞技+重社交的元素策略卡牌手游。

## 核心机制
- 叠印连锁系统
- 双轨制模式（竞技模式/派对模式）
- 四大元素体系（火、水、风、木）
- 海克斯系统

## 目录结构

```

├── 00_开发记录/          # 开发日志、迭代记录
```
